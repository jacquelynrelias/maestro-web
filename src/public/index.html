About

Maestro is a web and mobile experience to help you learn how to read sheet music in just four lessons. Our unified lessons gradually build on and reinforce each other so that you leave each session with a renewed sense of possibilities. We’ll take you through staff, rhythm, meter, and intervals. Meet your Maestro.

How

We use these world class technologies to bring Maestro to you. Our frontend is built using the Vue JavaScript framework. The backend is composed of a Parse server running on a MongoDB database. Our entire experience is run in Docker for hassle free development and deployment.
